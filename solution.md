#### Задание 1

Привести грамматику для языка палиндромов `{ w w^r | w \in {0, 1}* }`. Привести дерево вывода какой-нибудь строки длины не меньше 7 в предложенной грамматике. 

#### Решение

Грамматика:
```
G = <V_T, V_N, P, S>

V_T = {0; 1}
V_N = {W}
P = {S -> 0S0 | 1S1 | ε}
```

Дерево вывода для `11000011`:
```
      1   1   0   0
       \   \   \   \
root->  S - S - S - S - ε
       /   /   /   /
      1   1   0   0
```
